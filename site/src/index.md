# LiteIP Graph

Welcome to LiteIP Graph - a self-hosted Neo4j instance with Graph Data Science capabilities running on Cloudflare.

## Features

- **Neo4j 5 Community Edition** with Graph Data Science (GDS) library
- **AI Chatbot** powered by Cloudflare AI SDK
- **Graph Algorithms** for advanced analytics
- **Cloudflare Hosting** for global performance

## Quick Start

### Neo4j Connection

Connect to your Neo4j instance:

- **HTTP**: http://localhost:7474 (Neo4j Browser)
- **Bolt**: bolt://localhost:7687 (Application connections)

### AI Chatbot

Use the integrated AI chatbot to ask questions about graph databases, Neo4j, and Graph Data Science.
